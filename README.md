# camera.ui clients

[![License](https://img.shields.io/github/license/cameraui/clients)](LICENSE.md)

Client libraries for the camera.ui ecosystem — a typed transport layer and platform clients that talk to the camera.ui server over NATS-based RPC.

| Package | Description |
| --- | --- |
| [`@camera.ui/transport`](packages/transport) | Transport layer |
| [`@camera.ui/browser`](clients/browser) | Browser client |
| `@camera.ui/node` ([`clients/node`](clients/node)) | Node client — *planned* |

---

*Part of the camera.ui ecosystem - A comprehensive camera management solution.*
